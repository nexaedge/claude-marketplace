# Claude Marketplace

Jaison Erick's Claude Code plugin marketplace.

## Install

```
/plugin marketplace add claude-marketplace --source github --repo nexaedge/claude-marketplace
/plugin install spec-plugin@claude-marketplace
```

## Plugins

| Plugin | Version | Description |
|--------|---------|-------------|
| [spec-plugin](./plugins/spec-plugin/) | 3.1.0 | Full pipeline from ideation to verified code with 5 agents and 9 skills |
